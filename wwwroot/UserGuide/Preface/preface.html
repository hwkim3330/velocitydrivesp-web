Document Layout

This document describes how to use the VelocityDRIVE ™ Configuration Tool User's Guide configuration and programming tool for Microchip Switch products. The manual layout is as follows:

  • Chapter 1: "Overview" - This chapter explains the scope of the tool, what platforms it runs on, and several of its key features.
  • Chapter 2: "VelocityDRIVE™ CT Installation Instructions" - This chapter guides the user through VelocityDRIVE CT installation on both Windows and Linux
  • Chapter 3: "User Interface" - This chapter explains the first steps of creating new projects and how to navigate the user interface
  • Chapter 4: "Configurations in VelocityDRIVE™ CT" - This chapter outlines how to create, edit, and apply device configurations
  • Chapter 5: "Configuration Use Cases" - This chapter provides information on several common configuration use cases.
  • Chapter 6: "Command Line Tool Interface" - This chapter shows information on how to access the command line version of the tool
  • Chapter 7: "Default Configurations Backup Advisory" - This chapter contains necessary cautions to the user to avoid accidental mismanagement
  • Appendix A: "YANG Library View in Light or Dark Mode" - This appendix shows how the YANG library may be interpreted in both Light and Dark mode
  • Appendix B: "YANG File Paths for Typical L2 Settings" - This appendix provides information on various Layer-2 VLAN settings.
  • Appendix C: "Time Sensitive Networking Concepts" - This appendix shares information on various Time Sensitive Networking concepts that are relevant to the VelocityDRIVE CT's configuration capabilities.

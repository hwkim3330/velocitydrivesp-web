Overview

Introduction

The VelocityDRIVE ™ Configuration Tool (VelocityDRIVE CT) is a network configuration and monitoring software with a graphical user interface (GUI) designed to configure and monitor VelocityDRIVE SP-based switches. VelocityDRIVE SP is a turnkey L2 software application for Automotive Ethernet switches, built for constrained environments executing from on-chip memory.

Designed for both software developers and network implementers, VelocityDRIVE CT aides in the configuration and management of network switches through the customization of YANG module parameters. Its interface simplifies the complexities associated with YANG/CORECONF protocols, thereby enabling network developers to quickly generate exportable protocol configurations.

Developed as a cross-platform tool, VelocityDRIVE CT has been designed and tested on Windows and the Ubuntu distribution of Linux (it can be used with other Linux distributions and environments as well).

VelocityDRIVE™ CT Feature Summary

  • Modify or build custom configurations based on YANG catalog definitions
    • No coding required, configurations can be implemented entirely through GUI
    • Apply small updates incrementally or as full datastore replacements
    • Export or import configurations in CBOR or readable JSON
    • Element isolation feature to help with troubleshooting
  • YANG catalog integration and usability
    • Alongside each entry, the menu system provides contextual descriptions of YANG catalog IDs, definitions, statements, references, and deviations
    • Automatic download of up-to-date checksum verified YANG catalogs
    • YANG catalog code viewer, including display of module meta data
  • Connections available to multiple switches at once
    • Supports both MUP1 (UART) and IP-based connections
    • Real time monitoring of packet counters and operational states on each port
  • Log view, displaying a comprehensive list of system events, including tool-to-device commands and responses
  • Search tool for locating parameters in both configurations and connected devices
  • Help file, featuring an in-software version of the User Guide
  • Quick toggle for viewing values in either hexadecimal or decimal formats
  • Firmware update function
  • Light mode and dark mode options
  • Bundled command line tool

By integrating these features, VelocityDRIVE CT provides a solution that helps development and network teams deploy and manage VelocityDRIVE SP switches more effectively and quickly.

Introduction to the YANG Data Model

The YANG (Yet Another Next Generation) data model is a powerful and standardized data modeling language used to organize configuration and state data for information sent over network management protocols. Microchip has chosen to integrate the YANG/CORECONF standard into the VelocityDRIVE Software Platform and Configuration Tool. This decision is driven by several key advantages that YANG offers, making it an ideal choice for modern network management and configuration.

Why YANG?

YANG is a standardized data modeling language that is both human-readable and machine-friendly. This standardization is crucial as it ensures consistency and interoperability across different network devices.

YANG's modularity allows for the creation of model-driven tools that can be easily extended and reused. This modular approach not only streamlines our VelocityDRIVE toolset but also enhances the flexibility and scalability of network management solutions.

YANG supports various data encoding formats such as XML, JSON, and CBOR, making it versatile in different network environments. Additionally, YANG enables atomic transactions, ensuring that all changes are applied successfully or none at all, preventing misconfigurations and maintaining network stability. YANG has been widely adopted by key standards bodies such as the Internet Engineering Task Force (IETF) and the Open Networking Foundation (ONF). Furthermore, YANG's standardized approach reduces the need for proprietary extensions, thereby enhancing interoperability.

Additional Resources

For those interested in learning more about YANG and its applications, several resources are available:

  • IETF YANG Page: The Internet Engineering Task Force (IETF) provides comprehensive documentation and standards related to YANG. Visit the IETF YANG Page for more information.
  • Open Networking Foundation (ONF): The ONF offers resources and guidelines on using YANG in network management. Visit ONF YANG Resources for further details.
  • YANG Catalog: A repository of YANG models, including deviations and SID files, can be found at YANG Catalog. This resource is invaluable for developers and network engineers working with YANG.
